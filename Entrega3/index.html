API PARA GENERAR CODIGO FUENTE A TRAVES DE UN COMENTARIO

Existe una API de OpenAI, los mismo creadores de ChatGPT, mediante la cual podemos utilizar la inteligencia artificial, brindando como parámetro el mensaje/acción que deseamos realizar. Podemos indicarle que deseamos generar un código en el lenguaje que se desee.

Conclusion OpenAI es una inteligencia artificial que nos provee su API para utilizarla de manera gratuita, debemos registrarnos y obtener un API_KEY para utilizarla en cada consulta realizada. El usuario puede preguntar a la IA lo que desee y este devolverá una respuesta acorde.
